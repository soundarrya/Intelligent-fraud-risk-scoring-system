## Project Overview
Banks currently rely heavily on manual fraud detection, causing delays that lead to financial loss.  
This project introduces a real-time **risk scoring engine** to block or challenge suspicious transactions instantly.

---

##  Objectives
- Detect 80% of suspicious transactions automatically  
- Reduce manual review effort by 50%  
- Trigger OTP/FaceID for high-risk transactions  
- Block transactions with risk score ≥ 90  
- Deliver alerts within 5 seconds

##  Features
- Real-time fraud detection  
- Risk scoring (0–100)  
- Device, IP, and location mismatch detection  
- Alerts on medium and high-risk transactions  
- OTP/FaceID authentication for high-risk cases  
- Auto card block when user clicks “Not Me”  
- Fraud Analyst dashboard for review  
- Auto-generated fraud complaints
